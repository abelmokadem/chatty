# Chatty Style Guide
Style guide built using [fractal](http://fractal.build/). Runs [csscomb](http://csscomb.com/) as a precommit hook to 
keep the scss tidy. 

## Getting started

Run the following commands:
``` bash
$ npm install
$ npm start
```

Visit `http://localhost:3000` to see the style guide.
